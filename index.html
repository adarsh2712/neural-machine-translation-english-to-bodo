Neural Machine Translation

English to BODO translation with state of the art neural network architectures.

What is our Project?

Neural machine translation (NMT) is an approach to machine translation that uses a large artificial neural network to predict the likelihood of a sequence of words, typically modeling entire sentences in a single integrated model.

Hardware

Intel(R) Xeon(R) CPU @ 2.20GHz, 13GB RAM, Tesla K80 on Google Cloud.

Tools

Tensorflow 2.0, Python 3.6, Flask, Javascript, HTML, CSS, AJAX.

Dataset

We have created a language dataset from various sources available online and offline
